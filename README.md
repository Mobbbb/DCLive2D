# DCLive2D

使用说明:

#### ·步骤一
 点击 ```code``` 按钮下的 Download ZIP 选项下载并解压

#### ·步骤二
 安装Node.JS - https://nodejs.org/en/

#### ·步骤三
 在当前文件目录下打开 cmd 或者 powershell，输入 ```npm start```
