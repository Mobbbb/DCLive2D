一、下载
 > 方式一： https://github.com/Mobbbb/DCLive2D/archive/refs/tags/v1.2.2.zip
            https://github.com/Mobbbb/DCLive2D/archive/refs/tags/v1.2.1.zip
 > 方式二：点击 ```code``` 按钮下的 Download ZIP 选项下载并解压
 > 方式三：https://pan.baidu.com/s/1DPawE6L0-TxGjc0Gxv2r1Q?pwd=dc23 (v1.1.0版本)

#### 二、安装
 > 安装Node.JS - https://nodejs.org/en/

#### 三、启动
 > 方式一：双击 ```start.bat``` 文件启动
 > 方式二：在当前文件目录下打开 cmd 或者 powershell，输入 npm start
 
#### · 如何更新至对应版本
 > 下载对应版本的压缩包，用压缩包文件覆盖旧文件即可
 > https://github.com/Mobbbb/DCLive2D/releases/download/v1.2.2/DCLive2D-without-static.zip (v1.2.2)
 > https://github.com/Mobbbb/DCLive2D/releases/download/v1.2.1/DCLive2D-without-static.zip (v1.2.1)
